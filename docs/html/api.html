Public API#

Purpose#

partialwrap provides wrappers for Python functions and external executables so that they can easily be partialised with functools.partial.

Subpackages#

wrappers

Wrappers to partialize functions so that they can simply be called as func(x).

std_io

Parameter writer and output reader functions
