Public API¶

Purpose¶

partialwrap provides wrappers for Python functions and external executables so that they can easily be partialised with functools.partial.

Subpackages¶

wrappers

Wrappers to partialise functions so that they can be simply called as func(x).

std_io

Standard parameter reader and writer functions as well as output reader functions, including substitution of tags #JA????# or names in files.
